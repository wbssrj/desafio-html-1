6 - HTML não é linguagem de programação

Muita gente se confunde, mas HTML é uma Linguagem de Marcação.

Diferente de C++ ou Python, o HTML não possui estruturas lógicas como loops ou condicionais complexas.

Nota: Use HTML para estruturar, CSS para estilizar e JS para programar.
